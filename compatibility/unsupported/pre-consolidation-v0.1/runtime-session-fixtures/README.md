# Unsupported Pre-Consolidation Runtime Session Fixtures

These session smoke fixtures cover the unsupported pre-consolidation Runtime
session API and project payload shape. They are not the active graph 0.1
collaboration/session fixture model.

Any `/v0/session` default-session alias examples in this directory are
historical unsupported material only. Current Runtime smoke fixtures and scripts
must use explicit `/v0/sessions/{sessionId}` paths, including
`/v0/sessions/default` for the default session.

Active collaboration wire fixtures live in `compatibility/v0.1/collaboration`.
